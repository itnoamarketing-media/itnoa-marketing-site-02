# ITNOA Marketing Website V2

This is the cleaned production version. It does NOT use Tailwind, so it avoids the Tailwind/PostCSS deployment issue. It uses:

- Next.js 14
- React 18
- Framer Motion
- Pure CSS premium dark UI

## Deploy
Upload these files/folders to GitHub, then redeploy on Vercel.

## Replace media
Logos are in `/public`.
Video blocks are currently stylized placeholders. When ready, we can replace them with MP4 or YouTube embeds.
